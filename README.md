# Bioinformatics Projects
This repository contains basic bioinformatics projects implemented using Python.

# Projects

1. Multi-FASTA Sequence Analysis
. Reads multiple DNA sequences from a FASTA file
. Calculates nucleotide counts (A, T, G, C)
. Computes GC content

2. DNA Sequence Analysis Tool
. Takes DNA sequence as input
. Calculates sequence length
. Finds nucleotide frequency
. Computes GC content
